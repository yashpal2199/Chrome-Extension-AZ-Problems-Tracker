# Chrome Extension: AZ Problem Tracker  

## 🚀 Introduction  
**AZ Problem Tracker** is a Chrome extension that helps students bookmark and manage coding problems on [Algozenith](https://maang.in) platform.

With this extension, you can:  
- 🔖 Bookmark problems for later review.  
- 📂 View and manage saved bookmarks directly from the extension popup.  
- 💾 Persist data across sessions using Chrome Storage API.  

---

## ✨ Features  

-  **Bookmark problems** directly on [Algozenith](https://maang.in) for later review.  
-  **Real-time updates** with **JavaScript-based DOM manipulation** and content scripts.  
-  **Persistent storage** of bookmarks using **Chrome Storage API**.  
-  **Popup interface** for viewing and managing saved problems.  
-  **Manifest v3 support** ensuring secure and modern **Chrome extension architecture**.  
 

### Installation  

1. **Clone the Repository**  
   ```bash
   git clone https://github.com/yashpal2199/Chrome-Extension-AZ-Problems-Tracker.git
   cd Chrome-Extension-AZ-Problems-Tracker
2. **Load the Extension in Chrome**

- Open Chrome and go to: chrome://extensions/
- Enable Developer Mode (toggle in top-right).
- Click Load unpacked and select **Chrome-Extension-AZ-Problems-Tracker**

## How to Use  

1. Visit the [maang.in](https://maang.in) website.  
2. Use the AZ Problem Tracker extension to bookmark problems.  
3. Access your saved bookmarks from the extension popup.
